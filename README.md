🌐 Devfolio – MERN Portfolio & Admin Dashboard

Devfolio is a full-stack MERN portfolio website that began as a TechWise project and has grown into a live, personal platform for showcasing my work.
It includes an admin dashboard for uploading, editing, and managing projects, along with an interactive frontend that displays each project as a live preview.
The site is actively being updated as I continue to expand my portfolio.

✨ Features
🔑 User Authentication – Secure login & route protection with JWT and bcrypt.
📂 Project Management Dashboard – Upload, edit, and delete projects from an admin panel.
🖼️ File Uploads – Support for images, videos, .zip demos (extracted via Adm-Zip), and live project previews.
📊 Analytics Logging – Winston-based logging system to track activity and provide insights.
🎨 Frontend UI – Modern design with Tailwind CSS, shadcn/ui, Framer Motion animations, and responsive layouts.
📑 Project Previews – Interactive project demos open in 500×500 modal windows (e.g. HTML/CSS/JS mini-projects).
🌗 Dark / Light Mode – Custom theme toggle for better accessibility.
⚡ Fast Dev Environment – Powered by Vite for React frontend builds.

🛠 Tech Stack
Frontend: React.js, Vite, Tailwind CSS, shadcn/ui, Framer Motion
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication & Security: JWT (JSON Web Tokens), bcrypt.js
File Handling: Multer (uploads), Adm-Zip (ZIP extraction), File System (fs)
Logging & Monitoring: Winston Logger
Development Tools: Git, GitHub, Nodemon, Postman
Deployment (planned): GitHub Pages
